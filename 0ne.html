3) short note on tags,elements and attributes

Tags

Html tags are keyboard on a webpage that define how your web browser must format and display your web page. All tags contain opening tag and closing tag. Example: body tag, title tag, heading tag, table tag.

Elements

The html element is everything from the start tag to the end tag. consist of 3 parts: Opening tag, Content, Closing tag.

Attributes

An attribute is a piece of markup language used to adjust the behavior or display of an HTML element. For example, attribute can be used to change colour,size, or functionality of HTML elements. Examples: img src - src is a attribute, a href - href is a attribute.

9) Self closing tag? brief note on meta tags

A self closing tag in HTML is a kind of HTML tag that does not need to be closed manually by its closing tag, Which means it does not have a separate closing closing tag.

Meta tag

The meta tag defines metadata about an HTML document. Metadata is information about data. meta tag always go inside the head element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

10) What are global attributes? list any 5

Global attributes are attributes common to all HTML elements; they can be used on all elements, thiugh they may have no effect on some elements. Global attributes may be specified on all HTML elements, even those not specified in the standard.
