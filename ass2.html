8) Description list of full stack web development tech stack

MEAN STACK:
The MEAN(MongoDB, Express.js, AngularJS, and Node.js) is one of the most popular tech stack of 2023. The application of the MEAN stack are perfect for cloud hosting since they are flexible, scalable, and extensible. The stack can be deployed easily as it include its own web server.

MERN STACK:
The MERN stack is very similar to MEAN, in which React replaces Angular. Since React uses Virtual DOM, changes can be implemented easily. React is one of the most popular frameworks used for building high - end, single - page applications with interactive UIs. Since 2023 is all about enhancing UI, MERN stands to gain.

MEVN STACK:
In this version of MEAN, Angular is replaced by Vue.js as a front - end framework. This framework has shown immense popularity over the last few years. this functionality can be extended using third party services. Angular and React to give you great performance and a rich set of tools.

LAMP STACK:
The LAMP can be called a classic, time - tested stack of technologies - Linux, Apache, MySQL, and PHP. The LAMP stack can efficiently handle dynamic pages wherein the content changes every time is is loaded.LAMP being open source an non popularity. Linux as the OS, you can use Microsoft Windows to form a WAMP stack or MAC OS to form the MAMP stack.

9) Ordered list of the most common text formatting tags in HTML. Within each list item, use an unorderd list.

  1. Bold
    • The HTML bold tag specifies bold text, without any logical importance.
  2. Strong
    • The HTML strong tag is a logical tag that specifies text with importance. The content inside the strong element appears in the bold text but with semantic importance.
  3. Emphasizes
    • The HTML em tag emphasizes the text with semantic importance.
  4. Mark
    • The mark tag in HTML is used to highlight a text.
  5. Sup
    • The sup tag help us to superscript a text. Superscript text displays with a raised baseline using a smaller font.
  6. Small
    • The HTML small tag specifies smaller text. It is mostly used for copyright and legal text on a web page. With this tag, the size of the text will reduce onr font size more than the previous font size.
